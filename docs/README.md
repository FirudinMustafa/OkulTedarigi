- **[01-FINDINGS-REPORT.md](./01-FINDINGS-REPORT.md)** — 2026-04-08 sistem denetiminde tespit edilen tum bulgular

- **[03-NEXT-STEPS.md](./03-NEXT-STEPS.md)** — Production oncesi yapilacaklar listesi

- Yapilamayan ama gerekli olan isler **03-NEXT-STEPS.md**'ye yazilir
- Bulgu ID'leri (K1-K8 kritik, O1-O10 orta, S1-S12 kucuk) referansli olmali
